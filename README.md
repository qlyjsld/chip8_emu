# chip8_emu
chip-8 Emulator written in C++\
wiki: https://en.wikipedia.org/wiki/CHIP-8

<img src="https://github.com/wuhoipok/chip8_emu/blob/main/IBM.JPG" alt="IBM Logo">

# How to build
```
mkdir build && cd build
cmake .. && make -j32
```
